# Wedding Invitation Website (غانم ❤️ أمنية)

موقع دعوة زفاف إلكترونية احترافي فاخر ثلاثي الأبعاد بتصميم Glassmorphism (أبيض وذهبي) مع حركات سينمائية، عداد تنازلي، ورود متساقطة، لمعات ذهبية، ألعاب نارية عند نهاية الصفحة وموسيقى.

هذا المشروع جاهز للرفع المباشر والتشغيل على **GitHub Pages**.

---

## 📂 محتويات المجلد:
1. `index.html`: الهيكل الرئيسي وعناصر الصفحة والبطاقة باللغة العربية.
2. `style.css`: التنسيقات الفاخرة للزجاج والمؤثرات وألوان الذهب والأبعاد.
3. `script.js`: البرمجة الكاملة لـ Three.js (الظرف ثلاثي الأبعاد والفتح)، وGSAP للحركة السينمائية، وتساقط الورود واللمعات الذهبية والألعاب النارية والعداد التنازلي.

---

## 🛠️ كيف يمكنك تشغيل المشروع ورفعه على GitHub Pages:

### الخطوة 1: تنزيل ملفات المشروع
قم بتحميل ملفات هذا المجلد (`index.html` و `style.css` و `script.js`) على جهاز الكمبيوتر الخاص بك، وضعهم معًا في مجلد واحد (مثلاً باسم `wedding-invitation`).

### الخطوة 2: تغيير الأغنية والموسيقى (اختياري)
* بشكل افتراضي، يقوم الموقع بتشغيل ملف صوتي من الإنترنت كعينة تجريبية.
* لتشغيل أغنية **(هو حبيبي - أصالة)** أو أي ملف صوتي آخر من اختيارك:
  1. ضع ملف الـ MP3 الخاص بالأغنية في نفس المجلد بجوار ملف `index.html`.
  2. قم بتسميته مثلاً `music.mp3`.
  3. افتح ملف `index.html` وابحث عن السطر التالي:
     ```html
     <audio id="bg-music" src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" loop></audio>
     ```
  4. استبدل قيمة الـ `src` باسم الملف الخاص بك ليصبح هكذا:
     ```html
     <audio id="bg-music" src="music.mp3" loop></audio>
     ```

### الخطوة 3: الرفع على GitHub Pages
1. ادخل إلى حسابك في [GitHub](https://github.com/) وقم بإنشاء مستودع جديد (Repository) باسم مثلاً `wedding-invitation`.
2. اجعل المستودع **Public** (عام).
3. ارفع الملفات الثلاثة (`index.html` و `style.css` و `script.js`) والموسيقى `music.mp3` مباشرةً إلى المستودع.
4. اذهب إلى **Settings** (الإعدادات) الخاصة بالمستودع من شريط التبويبات العلوي.
5. من القائمة الجانبية اليسرى، اضغط على تبويب **Pages** (صفحات جيت هاب).
6. تحت قسم **Build and deployment**، اختر الفرع `main` أو `master` في حقل **Branch** ثم اضغط على زر **Save**.
7. انتظر دقيقة أو دقيقتين، وسيقوم جيت هاب بتوفير رابط موقع دعوة زفافك المباشر لتبدأ بمشاركته مع أحبائك وضيوفك!

## ✨ المميزات والتقنيات المستخدمة:
* **Three.js**: لتمثيل ظرف ثلاثي الأبعاد متحرك بإنارة ذهبية حقيقية وواقعية يفتح عند الضغط عليه.
* **GSAP Animations**: لعمل تسلسل انتقالي وحركة سينمائية عند فتح الظرف وخروج بطاقة الدعوة واقترابها من الشاشة ثم ظهور شاشة التفاصيل الفاخرة.
* **Tailwind CSS**: لتصميم وتنسيق بطاقة الدعوة وإعطائها طابع Glassmorphism الفاخر مع حواف ذهبية براقة.
* **إمكانيات مخصصة للجوال**: تصميم متجاوب بالكامل 100% مع الشاشات اللمسية ومقاسات شاشات الموبايل المختلفة.
* **مؤثرات بصرية متقدمة بالـ Canvas**:
  - تساقط بتلات الورود الوردية بشكل انسيابي فائق النعومة.
  - صعود ولمعان النجوم الذهبية الراقية.
  - إطلاق ألعاب نارية تملأ الشاشة عند النزول لأسفل الصفحة تأكيداً لبهجة الحفل.
* **مفاتيح التفاعل**:
  - زر موقع القاعة (قاعة البانسيه) المباشر على خرائط جوجل.
  - زر تأكيد الحضور المباشر عبر واتساب برقم العميل المخصص وبنص رسالة جاهزة.
