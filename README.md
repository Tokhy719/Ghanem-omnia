# Wedding-Invitation 💍

## غانم ❤️ أمنية

**التاريخ:** الجمعة 14/8  
**المكان:** قاعة البانسيه

## الرفع على GitHub Pages

1. ارفع المشروع إلى مستودع باسم مناسب (مثل `Wedding-Invitation`).
2. نفّذ البناء:

```bash
npm run build
```

3. من إعدادات المستودع → **Pages** → انشر محتويات مجلد `dist`  
   (أو استخدم GitHub Action لنشر Vite تلقائيًا).

> ملاحظة: إن كان المشروع على مسار فرعي (`username.github.io/repo-name/`)، اضبط `base` في `vite.config.ts` إلى `'/repo-name/'`.

## تجربة المستخدم

1. تفتح الصفحة → شاشة تحميل أنيقة.
2. يظهر **ظرف ذهبي ثلاثي الأبعاد** — اضغط لفتحه.
3. تفتح الطية وتخرج **بطاقة الدعوة** بحركة سينمائية.
4. من البطاقة أو الصفحة: **اللوكيشن** و **ضيوف (واتساب)** والعداد التنازلي.
5. زر الموسيقى العائم لتشغيل/إيقاف الأغنية.
6. عند التمرير لنهاية الصفحة تشتعل **الألعاب النارية**.
